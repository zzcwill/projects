查看征信状态

在系统页面下，点击客户管理，进入客户管理页面，可查看已发起征信是否通过。

点击可征信列表之后的“电话键”，进入呼叫客户页面，点击【呼叫】直接呼叫客户电话，点击【取消】，取消呼叫客户。

查看征信详情

在客户管理页面，点击征信列表的客户，可查看客户详情，在详情页面我们可以查看客户的征信状态及客户的征信更新时间。

查看征信详情

在客户管理页面，点击征信列表的客户，可查看客户详情，在详情页面我们可以查看客户的征信状态及客户的征信更新时间。

在客户详情，点击【征信记录】可查看客户的征信详情。

在征信客户详情页面，点击担保人之后的“v”,可查看担保人的征信是否通过。

添加新客户

在客户管理页面，点击右上角的“+”进入添加客户页面， 输入客户姓名、身份证号及手机号，其证件号码也可点击“绿色照相建“，将身份证正面放入相框中，系统可自动识别身份证号码，识别完成身份证自动录入证件号中。

贷款发起

在客户管理页面，如客户的征信状态“已通过”，点击“业务发起”进入业务发起页面，可发起贷款审批流程。

地图定位

在业务发起页面，输入客户基本信息及客户的贷款信息，地区选择填写客户详细地理地址，点击【地图定位】进入百度地图定位页面。

在百度地图选取客户的正确住址坐标，点击确定返回业务发起页面，坐标选取成功。

人员选择

在业务发起页面，如果客户的信息录入完毕，点击页面【下一步】，进入提交页面。

点击人员选择后的“>”在人员列表选择下一环节处理人，点选成功返回上一页面。

人员选择成功以后，点击页面下的【确定提交】，提交给下一环节处理人。
