征信发起

在工作台页面，点击右上角“按钮”选择【征信发起】，进入征信录入页面，录入客户信息，选择调查银行及客户是否已婚，添加客户担保人信息，确定无误点击页面下【确定提交】，征信发起成功，点击【返回】，返回上一页面。

添加担保人

在征信发起页面，点击添加担保人，进入添加担保人信息页面，输入担保人姓名，输入身份证信息，选择亲属关系，点击【确定】担保人添加成功。

亲属关系选择

在添加担保人页面，点选亲属进入选择亲属页面，根据担保人与客户的关系进行选择。

上传征信照片

在客户征信录入完毕，确定信息无误，点击页面下【确定提交】，进入征信图片上传页面。

在征信图片上传页面，点击“+”自动默认到手机系统相册中，选择客户身份证和授权书照片,选择完毕以后图片添加进来，会带有“未上传”标记。

征信图片都添加完毕，点击页面下的“上传”，图片开始进行上传，上传完毕后照片带有“已上传”标记，表示照片上传成功。

征信提交

征信照片上传成功后，在征信上传页面，点击【下一步】进入征信提交页面。

在提交页面，选择下一环节处理人，选择成功以后，点击【确定提交】，征信发起成功。
