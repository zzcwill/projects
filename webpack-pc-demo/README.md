# webpack
> A webpack4 demo

### 项目运行
- 安装相应依赖包
- yarn install
- 开发环境运行
- yarn dev
- 或 yarn dev2
- 生成项目打包文件
- yarn build
- 或 yarn build

### 1 webpack需要安装包
#### devDependencies

-	webpack-cli
